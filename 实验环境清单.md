## 3. ⚠ 实验边界（必须明确，否则会误读结论）

1. **测的是"抽取准确率"，不是"选型准确率"**：材质→源文件的映射当前在 `run_accuracy.py` 里**手工硬编码**（等于假设"选型已经选对"）。图纸解析→BOM→选型 是 M2 的活，**尚未接入**，本次实验**没有用任何生久图纸**。
2. **只用了选定供应商**：材料库里的冗余候选（线材的稳畅、端子胶壳的津达）**没用**。
3. **油墨（黑/白）跳过**：材料库里油墨只有 RoHS、无成分 MSDS（待向久益要）。
4. **真值不干净**：人工版承认书有错，准确率须对"源真值候选+久益签认"打分，不是盲对人工版。
5. 材料库其余约 80 份 PDF（各色 RoHS/SVHC/REACH/UL/盐雾/规格书等）**未用于本次准确率**。

## 4. 代码工作区

```
renshu/                 ← MVP 主包
  parse/   pdf_text(pdftotext/pdfplumber) + deid(本地脱敏)
  normalize/ units(weight/rohs/date/cas/supplier，已修3 bug)
  llm/     gateway(curl子进程) + schemas(MSDS/RoHS prompt)
  extract/ material(脱敏→LLM→清洗→JOIN)
  join/    material_join(成份记录)
  cli.py   tin-score(锡丝mock端到端)
  run_accuracy.py    ← M1 准确率跑批(MATERIALS定义在此) + .work缓存
  build_source_truth.py ← 双轨真值候选+差异清单
tests/
  golden/  答案key + 源真值候选 + 差异复核清单
  golden_diff.py  打分器(核心字段vs标签字段)
  unit/test_normalize.py  9/9绿
spike/   LLM抽取spike(锡丝100%) + ole_embed(OLE预研)
m7/      inject(数据注入) + migrate(OLE迁入)
模板/    黄金空白模板 + 注入/装配示例
.work/accuracy/  抽取缓存(PA66/PVC/磷青铜/聚烯烃/镀锡铜)
```

## 5. 实验 / 运行环境

| 项 | 值 |
|---|---|
| LLM | **qwen3.7-plus**，经 **aiapi.world 中转站**（OpenAI 兼容），curl 子进程**直连**（`--noproxy *`），`-m` 硬超时 |
| 数据边界 | 开发期**只送脱敏文本**（`renshu\parse\deid.py` 本地遮蔽供应商/报告号/公司名/机构/人名/电话/邮箱/地址）；过安全闸 |
| 运行时 | Python 3.14；pdftotext（Git/mingw64）；pdfplumber；PyMuPDF(fitz)；Excel COM(2007)+pywin32 |
| 环境变量 | `JIUYI_BASE_URL=https://aiapi.world/v1`、`JIUYI_API_KEY=<中转站key>`、`JIUYI_MODEL=qwen3.7-plus` |
| 复现命令 | `python renshu/run_accuracy.py PVC 镀锡铜 PA66 磷青铜 聚烯烃`（缓存命中则不重调 LLM）；`python renshu/build_source_truth.py` |
