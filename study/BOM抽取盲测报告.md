# BOM 抽取逻辑鲁棒性 · 7 案独立盲测报告

> 盲抽（官方 DashScope/qwen-plus，不看 golden）→ 按 CAS 归到各 golden 材质 → 逐字段对标。
> 源=承认书内嵌的 MSDS/报告 PDF；真值=各承认书材质表人工填的成分/CAS/重量。

## 方法
1. 从每份承认书材质表抽出全部唯一 OLE 源 PDF（本地）。
2. 文本层判为 MSDS 者送官方 qwen 抽成分（CAS/名称/重量），缓存避免重调。
3. 每份抽取按 CAS 重叠归到其 golden 材质（CAS 可区分同名金属在不同材质）。
4. 逐材质比 CAS 召回与重量一致。**对标人工 golden（已知含错，故区分源/料局限）**。

## 总体结果

| 指标 | 值 | 说明 |
|---|---|---|
| 覆盖 | 7 案 / 40 材质 | 参考承诺书集全量 |

| CAS 召回（已嵌 MSDS 料） | 131/141 = 92% | 含 6 处源缺(锡改性松香源里没有) |
| CAS 召回（全部 40 料） | 131/155 = 84% | 含 14 处源未嵌 MSDS |
| 真·抽取漏 | 4 处 | 源里有却没抽出（油墨异佛尔酮 78-59-1） |
| 源缺（源里没有该 CAS） | 6 处 | 非抽取错（锡改性松香未在源 MSDS） |
| 源未嵌该料 MSDS | 5 料 / 14 CAS | 非抽取错：只嵌 RoHS 报告 / 人工用外部CAS |
| 重量一致（非合金料） | 78/78 = 100% | PVC/油墨/聚烯烃等，几乎全中 |
| 重量一致（合金料） | 9/59 = 15% | 铜/锡/青铜：范围/代表值，人工取中值≠抽取错 |

## 结论
- **源里实含的 CAS，盲抽召回 97%（131/135，7 案 35 料）**——跨不同供应商/材料稳定。
- 真·抽取漏仅 4 处（油墨异佛尔酮 78-59-1，固定可复现，调 prompt 可补）。
- 其余漏配全是**非抽取因素**：源缺（锡改性松香源里没有）、源未嵌 MSDS（部分承认书只嵌 RoHS 报告）、人工用外部 CAS（PA66）。
- 重量：非合金料近乎全中；合金料人工取范围中值，与源范围值均有效，非抽取错。
- **结论：抽取逻辑鲁棒（源含即抽得到，~97%）；瓶颈在源完整性与人工 golden 口径，不在抽取本身**。
