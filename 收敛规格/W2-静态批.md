### 模板静态（工具不碰，但作"基线断言"自检其未被改动 / 防换错模板）
- **判定栏 E7–E27** = 非车规固定模式：E18–E22(项 8–12)=N/A、E26(14.3)=N/A、其余=V。
- 厂商 B30 = 久益；是否车规 D30 = 否；制定 E30 = Leven；核准 F30 = 王立均。
- 全部点检项文字、注解 A28、表头标签 = 静态。

### 怎么自检
- 读回 A30 == 料号。
- 静态基线断言：E7–E27 判定模式、B30/D30/E30/F30 == 上述期望值（变了→报警，可能换错模板或车规未识别）。
- 空白→填入 diff：变更格**恰好 = {A30}**。
- 无外链、media 不膨胀（W0 管线保证）。

### 可能的坑
- A30 与封面 D14、材质表表头（公式 `封面!D14`）必须同一料号 → 统一取图纸品号。
- 若来了**车规**产品：D30 应=是、E18–E22 应翻 V、sheet10 要填 PFMEA/控制计划等——本规格**不覆盖车规**（见下）。

### 怎么提醒用户 / 人工介入
- 是否车规 = 人工选（默认否）。选"是"→ 触发车规分支（暂未实现，提示"车规件需补 PFMEA 等，当前版本未支持，请人工处理"）。
- 静态基线断言失败 → 提示"模板异常/疑似换版，请核对"。

### 哪些手动填 👤
