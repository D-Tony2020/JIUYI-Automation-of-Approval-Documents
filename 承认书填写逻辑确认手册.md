**✅ governing 决策（已确认）**

1. **覆盖料类来源**：✅ **解析生久图纸「技术要求」段得到 BOM** → 映射久益中文料类。映射示例：`LEAD WIRE→导线/线材`、`HOUSING→胶壳`、`TERMINAL→端子`、`H/S TUBE→套管/热缩管`、`无铅焊锡/浸锡→锡`。
   - Sheet 3 覆盖料类 = 直接按技术要求列出的料类。
   - Sheet 6/8/9/11/12 = 暂按《附表-承认之细则》器件类别动态决定（⏳ 老板后续给更多承认书样本归纳 + 与久益确认）。
   - 🔑 **此即全流程 BOM 提取入口**，同时驱动"选型确认"与"材质成分表"。
2. **文件识别**：✅ 操作员先选供应商 → 系统在该供应商下**自动按类型分类并初选 → 人工复核**。
   - **MVP 输入侧简化**：操作员把本单要用的所有文件统一丢进一个**临时选取池文件夹**，系统从池里分类选取。
   - 自动分类命中率不足时 → 推动久益统一文件命名规范。
3. **多份处理**：✅ 同类型多份时**人工勾选**要嵌哪几份（系统列候选、操作员勾）。

## 🔑 关键归纳（8 份样本，影响全局架构）
1. **BOM 可变、由图纸技术要求驱动**：样本含 线材+锡(2行)、线材+端子(2)、线材+端子+锡(3)、线材+套管+锡(3)、线材+端子+套管+锡(4) 多种组合 → **不能假设固定 4 料**。

4. 签字人 8 份全部 制作=Leven/审核=王立均/核准=张启宇；材料名称 8 份全部"导线"（久益现仅做导线类）。
